## Python 코딩 지침

이 문서는 파이썬 코드 작성 시 준수해야 할 규칙과 지침을 설명합니다.

*  1. 파이썬 파일 수정 시 원본 파일을 직접 수정하는 대신, `write_file`을 사용하여 새 파일에 작성한다.
*  2. 여러 줄에 걸친 문자열(multiline string) 작성 시, 삼중 따옴표(triple quotes)를 사용한다.
*  3. 새 파일에 작성한 파이썬 파일은 다음 절차를 따른다:
    *   테스트를 진행하고 사용자의 확인을 받는다.
    *   기존 파일은 해당 디렉토리의 `bak` 서브디렉토리로 옮겨 놓는다.
    *   새 파일은 원본 파일 이름으로 변경(rename)한다.
